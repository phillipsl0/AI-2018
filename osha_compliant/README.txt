To run our code:

1. In terminal, go to the directory with OSHACompliantHeatMiser.py
2. Run the program by entering (without the quotation marks): "pipenv run python OSHACompliantHeatMiser.py"
    a. if you run into the following error: "RuntimeError: Python is not installed as a framework."
        -> echo "backend: TkAgg" >> ~/.matplotlib/matplotlibrc
3. Follow the printed prompts.
